Introduction to Python

Python is a high-level programming language... Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for: web development (server-side), software development, mathematics, system scripting.

Python Basics

Data types, variables, operators, and more...

In programming, data type is an important concept.

Variables can store data of different types, and different types can do different things.

Python has the following data types built-in by default, in these categories:

Text Type: str Numeric Types: int, float, complex Sequence Types: list, tuple, range Mapping Type: dict

Set Types: set, frozenset Boolean Type: bool Binary Types: bytes, bytearray, memoryview None Type: NoneType
